# hbskill 使用说明

## 这是什么

`hbskill` 是 Huberman Lab 播客访谈和科学健康主题的 Agent Skill 工具箱。它默认连接 `Huberman Lab 知识库`，帮助你学习资料、整理访谈、提炼协议、生成内容和制定行动计划。

## 适合谁

- 想系统学习 Huberman Lab 的用户
- 想把播客内容整理成笔记、内容或研究简报的人
- 想把睡眠、专注、压力、训练、营养等主题转成可执行计划的人
- 想用 IMA 知识库做来源检索和事实核对的人

## 安装方式

```bash
npx -y skills add FocusLiz-Lab/hbskill -g --all
```

## IMA 知识库配置

默认知识库：

```text
Huberman Lab 知识库
```

如果你还没有安装 IMA skill：

```text
请安装 ima 技能
下载地址：https://app-dl.ima.qq.com/skills/ima-skills-1.1.7.zip
API Key 获取：https://ima.qq.com/agent-interface
```

## 常用命令

```text
$hbs 用户想把这个知识库用于实际项目，帮他判断应该从哪个模块开始。
```

```text
$hbs-learning-map 系统学习这个知识库，先看哪些？
```

```text
$hbs-protocol 把睡眠、晨间光照和咖啡因管理整理成 7 天协议卡。
```

```text
$hbs-content 围绕知识库里的核心主题，生成 10 个内容选题。
```

```text
$hbs-research 整理多巴胺、动机和专注相关内容，输出证据表。
```

```text
$hbs-roadmap 做一个 30 天改善睡眠和专注的行动计划。
```

```text
$hbs-ima 搜索 physiological sigh 在知识库里的相关材料。
```

## 典型场景

学习路径：用 `$hbs-learning-map` 确定主题顺序和每阶段产出。

协议执行：用 `$hbs-protocol` 提炼协议卡，再用 `$hbs-roadmap` 做 7/30/90 天计划。

内容创作：用 `$hbs-research` 整理证据，再用 `$hbs-content` 生成选题、脚本或文章框架。

出处核对：用 `$hbs-ima` 搜索 IMA，再把结果交给对应 workflow。

## 常见问题

如果回答没有引用具体资料，通常是 IMA 未安装、未配置或当前账号没有知识库权限。先检查 IMA skill 和 API Key。

如果涉及补剂、药物、激素、妊娠、疾病、成瘾、精神健康或疼痛，请把输出当作学习资料和医生沟通清单，不要当作诊断或治疗建议。

## 发布边界

本项目不包含 Huberman Lab 原始转录稿、X/Twitter 原文、PDF 书籍或私有资料。具体引用和原文核对必须回到 IMA 知识库。
